RDCC Chat Client Configuration Instructions

(for the RICO project)


Overview

The RDCC chat system is running a standard IRC server.  There are many IRC chat clients, including mozilla.org's Chatzilla that will work with this server.  Once you have properly configured Chatzilla, you can simply enter in the URL irc://rdcc.guest.ucar.edu/%26RICO?pass=60kw60E08p to access the main RICO chatroom, for example.  We have found that xchat is also a good multi-platform chat client which offers enhanced features such as logging, timestamps, etc.  Since xchat running on a Windows platform has been the most popular method, we have included more detailed instructions for that type of install here.  There could be slight changes to what you see here due to version number differences, etc.  but you should also be able to configure your particular choice of chat client by extracting configuration information presented on this page and entering it appropriately.  Please pay particular attention to the chat server name, the chat server password and the conventions for chatroom names and nicknames. 



To Download and Install XChat on a Windows machine:

 

1)      With the browser of your choice, go to www.xchat.org

2)      On the left hand side of the page, click on Download

3)      Choose the Windows version under Precompiled Binaries

5)      Save to pc

6)      Open the .exe that was just saved to your pc

7)      Use all the defaults by clicking Next and then Install

8)      At the end it will ask you to click Next, and then Finish

9)      XChat is now installed and needs to be setup

10)  It should create an Xchat icon on your desktop

 

 

 

 

To Setup XChat


1)    Open XChat
2)    The first screen to display is the “XChat: Server List”
3)    We recommend that your Nickname will be the same as your Wiki/Discussion Forum login name, including the location you are currently chatting from, for example:
BrandonSlaten-ncar
SusanStringer-c130
MikeDaniels-spol
HarryOchs-ops
BjornStevens-ucla

4)   Also uncheck “Skip server list on startup” at the bottom of the screen

5)      Type in RDCC, case sensitive, so please use all caps
6)      RDCC should now show up in the Server list

7)      Click on Edit
8)      Edit default

 

 

 

10)  It will probably default to “newserver” of some sort.

11)  Replace ‘newserver” with “rdcc.guest.ucar.edu”

12)  Check “Connect to selected server only”

13)  Check “Use global user information”

14)  Check “Auto connect to this network at startup”

15)  Check “Use proxy server”

16)  Leave “ssl” choices unchecked

 

 

 




17)  You will also need to put in the channels you wish to join.  It is best to only use those you will really need.

18)  These are the established channels for RICO (more can always be added):
&RICO : RICO general discussion, aircraft and project coordination
&RICO-C130: C-130 specific discussion (instrumentation issues, etc.)
&RICO-Spol: S-Pol specific discussions
&RICO-dropsonde : dropsonde coordination
&RICO-BAE146: UK BAE-146 specific discussions
&RICO-Kingair: UWyo KingAir specific discussions
&RICO-SewardJohnson: Seward Johnson ship specific discussions
&RICO-PAM:  PAM station discussions
&RICO-GAOS: GAOS discussions

19)  No matter how many you choose, we recommend that you list “&RICO” as the last one.

20)  The Server password is 60kw60E08p

21)  All of the previous are case sensitive

22)  Now click on close

 

 

 

 

 

 

 

23)  You will be back at the “X-Chat: Server list”, click on connect to join the channels.

 

 

 

 

 

 

 

24)  The following commands will be very useful in the rooms:

Join a channel:
/JOIN <channel-name>
Change your nickname:
/NICK <new nickname>
Start a private chat with someone:
/QUERY <nickname>

25)  Please use the /JOIN command at the beginning to get into any of the rooms.

26)  You should now be able to chat!
