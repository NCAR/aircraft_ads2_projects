                       Aircraft Research Instrumentation
                          NCAR C-130 - Project #6-110
                       Coastally Trapped Waves Experiment
                               David Rogers
                                       

RAF-Supplied Instrumentation:

I.  Airborne Data System.
     A. Acquisition.  C-130 ADS2 (Motorola 68040 based). 
        NCAR Distributed Sampling Modules (4 units), one in
        each wing pod and two in cabin.
     B. Operator Station.  SunSPARC 10-402, PowerVideo Board, 
        16" High Resolution Color Monitor, Exabyte Model EXB-8505
        Tape Drives (2 units), 1-Gb External Disk (2), Sun CD--ROM
        Drive, Diamond Trackball, HP3630 PaintJet Printer, Powerware
        Prestige Model 800 UPS, 7" intercom control panel.
     C. Aft User Displays(2): SunSPARC 5 (2), PowerVideo Board (1),
        High Resolution Color Monitors (2), HP1200C/PS DeskJet
        Printer (1), Powerware Prestige Model 800 UPS (2).
     D. Flight Deck Display: Sun SPARC 5 (1), 16" High
        Resolution Color Monitor, HP1200C/PS DeskJet Printer, 
        Diamond Trackball, Powerware Prestige Model 800 UPS.

II.  Aircraft Position, Velocity and Attitude
     A. Honeywell Model HG1095-AC03 Laseref SM Inertial
        Reference System . (Research).
     B. Trimble TANS III Global Positioning System. (GPS).

III.  Static Pressures.
     A. Rosemount Model 1501 Digital Pressure Transducer -
        Fuselage Port (PSFD). 
     B. Rosemount Model 1201F1 Pressure Transducer - Fuselage
        Port (PSF).

IV.  Dynamic Pressures.
     A. Rosemount Model 1221F1VL - Radome (QCR). 
     B. Rosemount Model 1221F1VL - Fuselage Pitot (QCF). 

V.  Temperatures.
     A. Rosemount Type 102 Non-deiced Sensor -- Rosemount Model
        510BF Amplifier - Starboard Radome Mount (TTRR). 
     B. Rosemount Type 102 Non-deiced Sensor -- Rosemount Model
        510BF Amplifier - Port Radome Mount (TTRL). 
     C. Rosemount Type 102E Deiced Sensor -- Rosemount Model
        510BF Amplifier - Left Inboard Wing Access Plate (TTWH).
     D. OPHIR-III -- Radiometric Temperature - Left Fuselage (ATO).

VI.  Flow Angle Sensors, Radome.
      A. Attack - Rosemount Model 1221F1VL Differential Pressure
         Transducer (ADIFR). 
      B. Sideslip - Rosemount Model 1221F1VL Differential 
         Pressure Transducer (BDIFR). 

VII.  Dew Point and Humidity.
     A. General Eastern, Model 1011B Dew Point Hygrometer -         
        Starboard Radome Mount (DPT). 
     B. General Eastern, Model 1011B Dew Point Hygrometer -         
        Starboard Radome Mount (DPB). 
     C. NCAR Model LA-3 Stub Lyman-alpha Hygrometer - Port     
        Radome Mount (VLA).  
     D. NCAR Model LA-3 Crossflow Lyman-alpha Hygrometer - Port
        Radome Mount (VLA1).  

VIII.  Cloud Physics.
      A. Rosemount Model 871FA Icing Rate Detector - Right
         Inboard Wing Access Plate (RICE). 
      B. PMS Liquid Water Sensor (King Probe) -
         Left Outboard Wing Access Plate (PLWC).
      C. PMS Liquid Water Sensor (King Probe) -
         Right Outboard Wing Access Plate (PLWC1).
      D. Particle Measuring Systems Model PCASP - Right Pod
         Size Distribution, 0.12 to 3.12 um.
      E. Particle Measuring Systems Model FSSP/100 - Right Pod
         Size Distribution, 3 - 45 um.
      F. Particle Measuring Systems Model 260X - Right Pod
         Size Distribution, 10 to 640 um.                        
      G. Particle Measuring Systems Model 2D-C - Left Pod
         Size Distribution, 25 to 800 um.
      H. Particle Measuring Systems Model 2D-P - Left Pod
         Size Distribution, 100 to 3200 um.
      I. TSI Model 3760 CN Counter (CONCN).
         Size Distribution, 0.01 to 3 um.

IX.  Radiation Fluxes.
      A. Visible Radiation.  RAF Modified Eppley Model PSP
         Pyranometers - 2 units: Upward looking (SWT), Downward
         looking (SWB). Multipurpose Radiometer Housings.
      B. Ultraviolet Radiation.  RAF Modified Eppley Model TUVR
         Pyranometers - 2 units: Upward looking (UVT), Downward
         looking (UVB). Multipurpose Radiometer Housings. 
      C. Infrared Radiation.  RAF Modified Eppley Model PIR
         Pyrgeometers - 2 units: Upward looking (IRT), Downward
         looking (IRB). Multipurpose Radiometer Housings.
      D. Remote Surface Temperature.  Heiman Model KT19.85 
         Radiometer - 2 units: Both down looking (RSTB,RSTB1). 

X.  Geometric Altitude. 
      A. Steward-Warner Model APN-232(v) - Fuselage Mount,
         0 to 15,000 m AGL (HGM232). 

XII.  Photography.
      A. Sony XC-999 S-VHS Color Camera with JVC Model HR-S4700U
         Super VHS Recorder (3 pairs), Horita GPS Video Titler,
         Forward facing Cockpit Mount; Left Side-Facing Fuselage
         Mount; Down-looking Mount.

XIII.  Special Instrumentation.
      A. SABL Dual Wavelength Lidar.
      B. LORAN Dropsonde
      C. TECO Model 49 Ozone
